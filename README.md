## üì¶ Deployment Instructions (Vercel)
1. Fork or upload this repo to your GitHub.
2. Connect the repository to Vercel.
3. In Vercel Project Settings, add the following Environment Variables:
   - `SUPABASE_URL`: Your project URL from Supabase dashboard.
   - `SUPABASE_ANON_KEY`: Your project's anonymous key.
